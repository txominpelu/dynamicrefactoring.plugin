Installation and Configuration

Installing this refactoring plugin is quite an easy task. You will have to make sure though, that the destination system fulfils all the specified hardware and software requirements in order to be able to run the plugin correctly. Once you have done so, you can install the plugin onto your Eclipse installation by just following these steps:

  • Go to your Eclipse installation directory, and look for a folder named "plugins".
  • Copy the folder with the name "DynamicRefactoring_2.0.0" that you will find with the plugin distribution that you got to that "plugin" folder.
  • Restart your Eclipse.
