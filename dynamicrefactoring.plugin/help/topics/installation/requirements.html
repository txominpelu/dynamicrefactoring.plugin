Hardware and Software Requirements

This plugin should be able to run on any laptop or desktop computer which supports Eclipse. However, its performance, as well as Eclipse's will be increased the better the processor and the greater the available RAM are. Running the Java Virtual Machine is a heavy duty, so you will need a powerful machine in order for it to be run efficiently.

It is thus recommended not to install the plugin on computers that not fulfill the following minimun requirements. Otherwise it cannot be guaranteed the application will work optimally.


Component Required Recommended
Operating System Windows Windows XP Professional
Processor Pentium (or equivalent). Pentium IV or superior.
RAM 512 MB 1 GB
Disk space 250 MB 300 MB
Java JRE 1.5 JRE 1.6
