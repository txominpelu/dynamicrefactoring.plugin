Exporting Refactorings

In this section you will learn how to export your existing dynamic refactorings so that you can easily install them onto any other Eclipse installation with the Dynamic Refactoring Plugin.

  1. Inside Eclipse select the menu item Dynamic Refactoring > Export Refactoring... to open the Dynamic Refactoring Export dialog.




    Select Export from menu


    A list with the available dynamic refactorings will show up.



    Available refactorings


  2. Select the refactorings you would like to export. If you wish to export more than one dynamic refactoring, use the CTRL key at the same time as you select the refactorings.



    Select the refactorings you would like to export


  3. Push the "Examine" (...) button and select the output folder you want to export the refactoring files to.

    A new folder will be created under that directory for each of the exported refactorings.




    Select output folder


  4. After a short while, a confirmation message will open up telling you how many refactorings were exported.

    You can now find the exported refactorings under the output folder you selected.




    Confirmation
