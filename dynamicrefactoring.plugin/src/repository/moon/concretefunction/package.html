Proporciona las funciones básicas de acceso a la información de un modelo MOON.

Implementa un nivel de complejidad de acceso superior al que proporcionan las funciones accesoras propias de los objetos de MOON, permitiendo obtener colecciones o valores que cumplan restricciones más precisas.
