Paquete contenedor de los repositorio de acciones, predicados y funciones utilizados en la composición de refactorizaciones.

Además de los elementos concretos del repositorio, define una interfaz para el seguimiento de los mensajes distribuidos desde las acciones concretas durante su ejecución, lo que permite registrar "observadores" que sean notificados cada vez que se da un paso en el proceso.

Package Specification

El kit de compatibilidad de Java no necesita ninguna especificación para este paquete.
