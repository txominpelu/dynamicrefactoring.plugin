Implementa operaciones relacionadas con el manejo de ficheros.

Proporciona funciones que permitan efectuar las operaciones más habituales sobre ficherso y directorios, como copiarlos, crearlos o eliminarlos.

Los métodos ofrecidos tienen todos acceso a nivel de clase, es decir, como métodos estáticos.
