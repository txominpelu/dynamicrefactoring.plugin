Proporciona la implementación de las vistas del plugin.

Todas las clases del paquete deben extender e implementar completamente la clase abstracta org.eclipse.ui.part.ViewPart, que proporciona una implementación base para todas las vistas.

Las vistas implementadas se pueden enganchar al entorno de Eclipse mediante la configuración asociada en el fichero del plugin plugin.xml.

El kit de compatibilidad de Java no necesita ninguna especificación para este paquete.
