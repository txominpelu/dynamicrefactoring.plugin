Implementa listeners comunes a las interfaces gráficas de varias de las páginas del asistente de creación y edición de refactorizaciones.

Permite asociar comportamiento a botones de reordenación de listas seleccionables. Basta con asociarle una instancia del Listener apropiado a cada botón para que enganche con la función adecuada: mover arriba o mover abajo.
