Implementan la interfaz gráfica del plugin.

Las clases contenidas en el paquete raíz implementan algunas funciones concretas pos si mismas. Otras aportan funcionalidad gráfica común a varios de los elementos de la interfaz.

La implementación utiliza la biblioteca SWT de Eclipse, empleada para la construcción de la interfaz del propio IDE, por lo que se ha utilizado en lugar de AWT y Swing de Java.

Package Specification

El kit de compatibilidad de Java no necesita ninguna especificación para este paquete.
