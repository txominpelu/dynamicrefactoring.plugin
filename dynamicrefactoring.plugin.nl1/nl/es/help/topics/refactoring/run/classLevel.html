Ejecutar Refactorizaciones: Operaciones con Ámbito de Clase


Cuando se haya seleccionado una clase en el árbol desplegable de la vista "Package Explorer" u otra vista de Eclipse se activarán las refactorizaciones con ámbito de clase. Para ejecutar cualquiera de ellas, existen tres posibilidades.

  • Haga clic con el botón derecho en la clase sobre la que quiere aplicar la refactorización, y seleccione la opción Refactorizacion Dinamica > Ambito De Clase en el menu desplegable que aparece.


    menu desplegable: Ambito De Clase


  • Seleccione la opción de menu Refactorizacion Dinamica > Refactorizar > Ambito De Clase en Eclipse.


    menu:Ambito De Clase


Una vez que se indica que se quiere ejecutar una refactorización sobre el elemento seleccionado, se empieza a construir el modelo que se utilizará. Cuando el proceso termine, se mostrará una ventana de selección con la lista de refactorizaciones disponibles con ámbito de clase.


Refactorizaciones con ámbito de clase


En este momento debe elegir la refactorización que desea aplicar. Una vez que la haya seleccionado, pulse el botón Aceptar para iniciar el proceso.

  • Cuando se selecciona una clase de las vistas de Eclipse, la vista "Refactorizaciones dsiponibles" actualiza su contenido con las refactroizaciones de ámbito de clase. Después se debe hacer un doble-click en alguna de ellas para comenzar su ejecución.


    vista "Refactorizaciones disponibles"
