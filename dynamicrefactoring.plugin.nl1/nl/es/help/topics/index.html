Plugin De Refactorización Dinámica



Está guía le ayudará a utilizar la aplicación. A lo largo de ella se explica en detalle cómo funciona el plugin y cómo se puede utilizar.

Puede utilizar este plugin para definir y construir sus propias refactorizaciones o para personalizar algunas de las que se incluyen por defecto. También es posible eliminar aquellas que no se consideren ya necesarias.

Una vez que se ha definido una nueva refactorización, se podrá ejecutar en Eclipse inmediatamente, ya que el plugin construye la lista de refactorizaciones disponibles dinámicamente en cada ocasión.

Si necesita importar o exportar refactorizaciones, podrá también hacerlo fácilmente a través de los asistentes que le ayudarán en el proceso.

También incluye la utilidad de exportar e importar un plan de refactorizaciones.

El plugin permite ahora categorizar las refactorizaciones, crear clasificaciones nuevas y visualizar todas las refactorizaciones de forma organizada en la vista del catálogo. Esta vista proporciona además filtrado de las refactorizaciones basándose en sus categorías, su nombre, su descripción o sus palabras clave.

Por último el proceso de creación de nuevas refactorizaciones es ahora más sencillo gracias a una ventana de selección de elementos mejorada que muestra información adicional sobre el elemento seleccionado y proporciona un sistema de búsqueda avanzado.
