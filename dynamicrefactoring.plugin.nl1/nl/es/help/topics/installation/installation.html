Instalación y Configuración

Instalar este plugin es una tarea muy sencilla. Tendrá que asegurarse, no obstante, de que el sistema cumple con todos los requisitos hardware y software especificados si quiere poder utilizarlo sin problemas. Una vez hecho esto, podrá instalar el plugin en su instalación de Eclipse siguiendo simplemente los siguientes pasos:

  • Vaya al directorio de instalación de Eclipse, y busque una carpeta con el nombre "plugins".
  • Copie la carpeta con el nombre "DynamicRefactoring_2.0.0" que encontrará en la distribución del plugin a esa carpeta "plugins".
  • Reinicie Eclipse.
