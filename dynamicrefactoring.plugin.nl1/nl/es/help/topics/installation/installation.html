Instalación y Configuración

Instalar este plugin es una tarea muy sencilla. Tendría que asegurarse, no obstante, de que el sistema cumple con todos los requisitos hardware y software especificados si quiere poder utilizarlo sin problemas. En esta nueva versión del plugin se puede beneficiar de la instalacón y actualización a través del menú de instalación de software de Eclipse. Podría instalar el plugin desde dicho menú:

  • Utilizando como repositorio de instalación la siguiente ruta: jar:http://refactoring-plugin.googlecode.com/hg/dynamicrefactoring.p2repository.zip!
  • Escogiendo la característica dynamicrefactoring.feature.
  • Reiniciando Eclipse al finalizar la instalación.
Para la actualización sólo sería necesario acceder nuevamente al menú de ayuda de Eclipse pero esta vez pulsar la opción de comprobar actualizaciones y seleccionar la actualización de la característica dynamicrefactoring.feature ya citada.
